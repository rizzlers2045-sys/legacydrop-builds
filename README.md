LegacyDrop is a project of mine which semi-backports an AirDrop-like system to iOS 6 (and possibly down to iOS 4, but untested).
Sadly, it doesn't support sharing from an iDevice on iOS 6 to a newer iDevice (such as an iPhone, iPod touch, Mac, etc.) and probably never will (technical differences).
